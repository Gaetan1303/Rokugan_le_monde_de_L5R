# Adresses IP sortantes Render

## Informations réseau

Les requêtes réseau de votre service vers l'internet public proviendront de l'une des adresses IP ou plages suivantes.

**Important :** Ces IPs ne sont pas uniques à votre service ou espace de travail. Elles sont partagées par d'autres services Render dans la même région.

## Liste des adresses IP

### Adresses individuelles
- `18.156.158.53`
- `18.156.42.200`
- `52.59.103.54`

### Plages d'adresses (CIDR)
- `74.220.51.0/24` (74.220.51.0 - 74.220.51.255)
- `74.220.59.0/24` (74.220.59.0 - 74.220.59.255)

## Utilisation

Ces adresses IP doivent être utilisées pour :
- Configuration de whitelists sur des services externes
- Règles de pare-feu pour autoriser les connexions sortantes
- Configuration de sécurité réseau
- Restriction d'accès à des APIs tierces

## Région

Ces adresses correspondent à la région AWS `eu-central-1` (Francfort, Allemagne) basée sur les préfixes IP.
